My time at Great learning i learnt the basics of UI/UX which includes, UI(User Interface) which entails the visual and interactive elements of a digital product that users directly interact with.

Consistency in design is crucial. Elements should look and behave the same way throughout the product to reduce user confusion.
Prototyping: Creating prototypes helps in testing and refining the UI design before development, saving time and resources.
User Research: UX design starts with understanding the users and their needs. This involves conducting research, interviews, surveys, and analyzing user behavior to gather insights.

User Personas: UX designers often create user personas, fictional representations of typical users, to guide design decisions based on user goals and pain points.

Information Architecture: UX design involves organizing content and features logically to ensure users can easily find what they're looking for. This includes creating navigation structures and hierarchies.

Wire framing: Before designing the visual interface, UX designers create wireframes, which are low-fidelity sketches or prototypes, to outline the layout and functionality of the product.

Usability Testing: UX designers conduct usability testing to gather feedback from real users and make improvements based on their experiences.

User Flows: Designers create user flows to map out the journey a user takes through the product, from entry to completion of tasks.

Iterative Design: UX is an iterative process. Designers continually refine and improve the product based on user feedback and data.
